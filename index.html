HideNSeek.js

A jQuery plugin

What it is:

An interactive jQuery plugin, HideNSeek.js lets users turn the lights out and explore your site using a flashlight. But be careful while you're looking around, when the lights are off, the monsters come out...

How to use it:

1. Include the 'hidenseek.js' file in your scripts folder.

2. Link the 'hidenseek.js' script to your HTML file.

3b. 'hidenseek.js' accepts 3 values as follows:

Parameter Format Notes
Flashlight Size Any integer with a positive value Determines the diameter of the flashlight in pixels.
Darkness Value between 0-1 Determines the opacity of the flashlight shadow.
Gremlins Boolean If true, a gremlin will live on the page when the lights are turned off.
